-- Pixels by Jumpei --

Long ago, back in maybe 2004, I decided I would try my luck searching for a job as an artist for videogames, so I begun practicing pixel art. These characters are examples from that period.

"What in the sands name could that be??" "I dunno but he's getting too darn close to me..." "Maybe we're already dead and he's just letting us know that..." /06

"I tried to restrict myself in terms of size too, so I kept things small, also used #3B2E30 for contrast. A fun simple piece." Features what is maybe the first character I ever created, Esper.

More Jumpei's Shipyard characters, a conjurer and a runnaway thief, both wanted by law for different reasons.

I did these characters to try to materialize an old idea for a game called Hacktone, some hacker guy who would hack old hardware using music.The main character would be an hacker rocking a jumpsuit and a samurai sword (so 90's).

A simple gnome with a pipe. Anyone remember a cartoon called David the Gnome? At least here (europe) I used to watch as it a kid...So yeah, not entirely based on that but somewhat inspired by it.

So this was made for the Pixel Dailies carving theme. I'm not very skilled in animation, but I like how it came out.
